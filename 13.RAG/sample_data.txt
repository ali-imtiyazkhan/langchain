LangChain is a framework for developing applications powered by language models. It enables applications that are:
- Data-aware: connect a language model to other sources of data
- Agentic: allow a language model to interact with its environment

The core components of LangChain include:
1. Models: Supported models and integrations.
2. Prompts: Prompt management, optimization, and serialization.
3. Memory: State that is persisted between calls of a chain/agent.
4. Indexes: Structuring documents to LLMs can best interact with them.
5. Chains: Sequences of calls (whether to an LLM or a different utility).
6. Agents: Where the LLM decides which actions to take, takes those actions, observes the result, and repeats.
7. Callbacks: Logging and streaming intermediate steps of any chain.
